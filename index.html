GitHub အသုံးပြုနည်း

User: Thu Aung

ယခု GitHubအသုံးပြုနည်း Commandများသည် ကျွန်တော် Thu Aung ကိုယ်တိုင်သုံးရန်အတွက်သာရည်ရွယ်ပြီးမှတ်သားထားခြင်းဖြစ်သောကြောင့် GitHubနဲ့ပတ်သက်တာအားလုံးပါဝင်ခြင်းမရှိပါ၊ လိုအပ်လျှင်တော့သုံးနိုင်ပါသည်။

(Windows Computerမှာ GitHub အကောင့်အဟောင့်နှင့်ချိတ်မိလျှင် ဖြုတ်ပြီး GitHubအကောင့်သစ်နှင့်ချိတ်ဆက်နည်း)

  • 1. Computer SearchBoxတွင် Credential Manager လို့ရိုက်ရှာပါ။
  • 2. Windows Credentials ဆိုတာကို ရွေးပါ။
  • 3. အောက်နားက စာရင်းတွေထဲမှာ git:https://github.com ဆိုတာကို လိုက်ရှာပါ။ (တချို့စက်တွေမှာ GitHub ဆိုပြီး စတာလည်း ဖြစ်နိုင်ပါတယ်)။
  • 4. တွေ့ပြီဆိုရင် ဘေးက မျှားလေးကိုနှိပ်ပြီး Remove (သို့မဟုတ်) Delete ကို နှိပ်လိုက်ပါ။
  • 5. ဒါဆိုရင် ကွန်ပျူတာက အကောင့်ဟောင်းချိတ်ဆက်ထားတာပြုတ်သွားပါပြီ။
  • 6. GitHub အကောင့်သစ်နှင့်ချိတ်ဆက်ဖို့အတွက် အောက်ပါ Command နှစ်ကြောင်းကို ရိုက်ထည့်ပြီး Brower LogInဝင်ပေးပါ။
(git config --global user.name "မိတ်ဆွေရဲ့ GitHub နာမည်အသစ်")
git config --global user.name "Thu Aung"
(git config --global user.email "မိတ်ဆွေရဲ့ GitHub အီးမေးလ်အသစ်")
git config --global user.email "yelinnaung.official@gmail.com"

(1) Project အသစ်တင်နည်း

git init
git add .
git commit -m "First commit"
git branch -M main
git remote add origin your_github_repo_link
git push -u origin main

(2) တစ်ပိုင်းချင်း Updateတင်နည်း

git add .
git commit -m "Added profile page UI"
git push

(3) Version အသစ်တင်နည်း

git tag v1.0
git push origin v1.0

(4) ချိန်းထားတဲ့ Codeအဟောင်းတွေကို ပြန်စစ်နည်း(Terminalမှာရိုက်ပါ)

(Code Updateအပြောင်းအလဲတွေကို စစ်ဆေးတာ)
git log
(လိုချင်တဲ့ရက်ကCodeကိုအပြီးပြန်သွားတာ)
git reset --hard f4g5h6j
(လိုချင်တဲ့ရက်ကCodeကိုခနပြန်ကြည့်တာ)
git checkout <Day_2_ID>
(လက်ရှိCodeကိုပြန်လာတာ)
git checkout main

(5) Git Project ဒေါင်းနည်း

(Project Cloneလုပ်နည်း)
git clone git_repo_link write_your_folder_name
(Git ပေါ်က Cloneလုပ်ထားတဲ့ Codeတွေကို git reset နဲ့ဖျက်မိသွားရင်၊ Cloneပြန်လုပ်စရာမလိုပဲ Codeပြန်ထည့်နည်း)
git pull
(အပေါ်ကCommandနဲ့မရခဲ့လျှင်)
git pull origin main
